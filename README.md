# 🎬 Netflix Data Analysis Dashboard

## 📌 Project Overview

This project analyzes Netflix Movies and TV Shows dataset using Power BI to generate business insights.

## 🛠️ Tools Used

* Power BI (Dashboard & Visualization)
* SQL (Data Analysis)
* Excel (Data Cleaning)

## 📊 Dashboard Features

* KPI Cards (Total Titles, Movies, TV Shows)
* Content Distribution (Pie Chart)
* Content Growth Over Time (Line Chart)
* Top Countries by Content
* Top Genres on Netflix
* Interactive Filters:

  * Type (Button)
  * Year (List)
  * Country (Search)

![Dashboard](dashboard.png)

## 🚀 Key Insights

* Content increased rapidly after 2015
* Movies dominate Netflix catalog (~95%)
* Top countries contribute majority of content
* Drama and international genres are most popular

## 💼 Use Case

This project demonstrates how raw data can be transformed into actionable business insights using data visualization tools.
